How It Works
  1. Upload PDF: Select a medical PDF (treatment guidelines, ICD codes, etc.)
  2. Auto-Processing: PDF is split into chunks and embedded
  3. Index Update: Chunks are added to FAISS vector store
  4. Immediate Use: Chatbot can now retrieve info from this PDF
  5. Persistent: Index is saved and survives restarts
